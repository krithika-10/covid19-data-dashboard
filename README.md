# COVID-19 Global Analysis Dashboard

An interactive data visualization dashboard for exploring global COVID-19 trends — built with HTML5, CSS3, and JavaScript.

🔗 **Live Demo:** [https://krithika-10.github.io/covid19-data-dashboard/](https://krithika-10.github.io/covid19-data-dashboard/)

---

## About the Project

This dashboard was designed to make global COVID-19 data accessible and visually engaging. The goal was to allow anyone — regardless of technical background — to explore pandemic trends across countries, compare mortality and recovery rates, and understand the scale of the crisis through interactive visuals.

As a first-year BCA Data Science student, I directed the design and data decisions for this project, choosing what to visualize, how to structure the user experience, and which metrics matter most for public health storytelling.

---

## Features

- 🗺️ **Interactive World Map** — built with D3.js, allowing users to hover and explore country-level data visually
- 🔍 **Dynamic Country Slicer** — filter and deep-dive into any country's COVID-19 statistics
- 📊 **Real-time Data Visualization** — mortality rates, recovery rates, and case trends displayed through Chart.js
- 🎨 **Cyberpunk / Dark UI** — custom CSS3 theme designed for readability and visual impact
