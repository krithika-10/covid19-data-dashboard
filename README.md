Project Title: COVID-19 Global Analysis Dashboard.

Live Demo: https://krithika-10.github.io/covid19-data-dashboard/

Technologies Used: HTML5, CSS3 (Cyberpunk/Dark UI), JavaScript, Chart.js, D3.js.

Features:

Interactive World Map (D3.js).

Dynamic Country Slicer for deep-dive analysis.

Real-time data visualization of mortality and recovery rates.

Data Source: covid-19 dataset from Kaggle.

AI Collaboration: This project was developed as a collaboration between human direction and AI assistance to explore advanced data visualization techniques.
